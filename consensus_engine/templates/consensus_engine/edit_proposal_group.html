Edit a group's detail

Proposals can be grouped under a single category, only one level of hierarchy is allowed to keep the process simple, and proposals can only exist in one group. Users can view and vote on the proposals within the groups they are members of. At the moment all users are members of all groups, but group membership will be refined in the future.

Proposal Groups have a name and a description, the name will appear on the task bar on the right and the description is used to provide members an idea of what the group is designed to do.

Note: Only the user which created the Group can edit it. Groups cannot be currently deleted.
